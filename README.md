# Rock Paper Scissors Game

![page-header](imgs/githubImg.png)

## Description 
This is simple web game of Rock Paper Scissors made with Javascript. 

## Features


- Score counter 
- Can be played both on PC or Mobile layout
- Win state and rules modal
